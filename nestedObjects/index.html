Nested Objects

Nested objects are objects that are inside of other objects. They are accessed using dot notation.

For example, if we have an object called person with a property called name, we can access the name property like this:

person.name

If the name property is an object itself, we can access its properties like this:

    person.name.first
    person.name.last

Nested objects can be used to represent complex data structures, such as a person's address or a product's inventory.

Here is an example of a nested object representing a person's address:

    const person = {
        name: "John Doe",
        address: {
            street: "123 Main St",
            city: "Anytown",
            state: "CA",
            zip: "12345
        }
    }
